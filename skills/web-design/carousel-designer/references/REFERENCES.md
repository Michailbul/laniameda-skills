## Brand / Style References

### SpaceX / Tesla Aesthetic DNA
- Stark white backgrounds with near-black text
- Ultra-condensed display type (all caps, tight tracking)
- Precise engineering tolerances in spacing (nothing arbitrary)
- Data presented as fact, not opinion — numbers dominate
- Black borders that feel like technical drawings
- No gradients, no drop shadows, no decorative flourishes
- Grid as structure, not decoration

### Reference: Linear.app cards
Pattern: 2px solid border + 8px offset shadow (no blur)
Use for: stack-grid cards, metric cards
Key visual: border and shadow color differ → creates depth without softness

### Reference: Vercel stats section
Pattern: giant number (hero size), tiny label above in mono, nothing else
Use for: metric-card slides
Key visual: the number IS the slide. Everything else is negative space.

### Reference: Stripe press / editorial
Pattern: left-aligned, 2-column, tight body copy with generous margins
Use for: problem/tension slides
Key visual: body copy never exceeds 60% slide width — the right 40% is empty
